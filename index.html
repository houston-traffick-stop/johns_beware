JOHNS BEWARE: HUMAN TRAFFICKING PROJECT

Is created as a 3 pronged approach to deter the purchasing of commercialized sexual exploitation, using BackPage fake postings, a web browser tracking extension, and geo mapping

Fake_Posting

Deterring the Johns

Every week BackPage hosts over 16,000 ads commercializing sex trafficking, in Houston alone. Using fake posts, Johns will be informed of the consequences of their actions after clicking on BackPage ads. These fake posts will be generated and posted using a natural language processing model built on the scraped ads.

FRONT_PAGE

Chrome Extension Supporting Investigators (used by public and investigators)

This extension will:
Scan web pages for suspicious activity
Tracks and records data to database managed by local/national Police Departments
All data will be used for investigation purposes
Processing model built will gather location information, age and phone number (if available)
